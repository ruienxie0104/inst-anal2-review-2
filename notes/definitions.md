# 定義速查表

## Ch20 質譜法

| 英文 | 中文 | 定義 |
|------|------|------|
| Mass Spectrometry | 質譜法 | 根據荷質比分析離子的技術 |
| Ion Source | 離子源 | 產生原子/分子離子的裝置 |
| Hard Ionization | 硬離子化 | 高能游離，碎片多，分子離子峰弱 |
| Soft Ionization | 軟離子化 | 低能游離，碎片少，分子離子峰強 |
| Base Peak | 基峰 | 質譜中最高訊號峰，訂為100% |
| Isotope | 同位素 | 同元素，質子數同中子數不同 |
| Adduct | 加成物 | 不同分子互相加成的產物 |
| Isomer | 同分異構物 | 分子式同結構不同 |
| CID | 碰撞誘導解離 | 氣相中誘導選定離子碎裂的技術 |
| Precursor Ion | 母離子 | MS/MS中被選定碎裂的離子 |
| Product Ion | 子離子 | 母離子碎裂後產生的離子 |
| Resolution | 解析度 | 分辨相近質量離子的能力 R=m̄/Δm |

## Ch22-25 電分析化學

| 英文 | 中文 | 定義 |
|------|------|------|
| Oxidation | 氧化 | 失去電子，氧化數增加 |
| Reduction | 還原 | 得到電子，氧化數減少 |
| Cathode | 陰極 | 還原反應發生的電極 |
| Anode | 陽極 | 氧化反應發生的電極 |
| Standard State | 標準狀態 | 1M, 1atm, 100%純度的參考條件 |
| Standard Electrode Potential | 標準電極電位 | 相對SHE的還原電位 |
| Nernst Equation | 能斯特方程式 | 非標準狀態下的電極電位計算 |
| Voltammetry | 伏安法 | 通以時變電位量測電流的方法 |
| Amperometry | 安培法 | 固定電位量測電流的方法 |
| Working Electrode | 工作電極 | 施加電位差並量測電流 |
| Counter Electrode | 輔助電極 | 導通迴路，平衡電荷 |
| Reference Electrode | 參考電極 | 校正工作電極電位 |
| SHE | 標準氫電極 | E°=0.000V的參考基準 |
| SCE | 飽和甘汞電極 | E°=+0.244V，穩定但有毒 |
| Ag/AgCl | 銀-氯化銀電極 | E°=+0.197V，小型穩定 |
