# Ch20 質譜法 (Mass Spectrometry)

## 歷史

| 年份 | 人物 | 貢獻 |
|------|------|------|
| 1803 | John Dalton（道耳吞） | 提出 H=1，Dalton (Da) 原子質量單位 |
| 1860 | Cannizzaro（坎尼扎羅） | 以 O=16 amu 為基準的原子質量單位 |
| 1919 | Aston（阿斯頓） | 發明第一台質譜儀，1922 諾貝爾獎 |

## 質譜儀基本原理

- **質譜法 (MS)**：根據**荷質比 (m/z)** 分析離子
- 流程：樣品 → 離子源（離子化）→ 質量分析器（分離）→ 偵測器 → 輸出

### 關鍵公式

- 離子加速動能：**KE = zeV**
  - z：電荷數，e：電子電量（1.6×10⁻¹⁹ C），V：加速電壓
- Example：z=1, V=10³V → KE = 1.6×10⁻¹⁶ J

## 離子源 (Ion Sources)

### 硬離子源 vs 軟離子源

| | 硬 (Hard) | 軟 (Soft) |
|---|---|---|
| 能量 | 高 | 低 |
| 碎片 | 多 | 少 |
| 分子離子峰 | 小或消失 | 大且明顯 |
| 結構資訊 | 豐富 | 較少 |
| 分子量 | 不易得知 | 容易得知 |

記憶：**硬碎軟全**

### 六種離子源比較

| 離子源 | 硬/軟 | 原理一句話 | 分子離子峰 | 串聯層析 | 特色 |
|--------|-------|-----------|----------|---------|------|
| EI | 硬 | 70 eV 電子束撞擊 | 弱 | GC-MS | 最常用硬離子源，碎片多結構清 |
| CI | 軟 | 試劑氣體先離子化再碰撞樣品 | 強 | GC-MS | 四種反應：質子轉移/加成/氫離子轉移/電荷轉移 |
| FAB | 軟 | 快速中性原子撞擊樣品 | 強 | 否 | 適用高極性、低揮發、熱不穩定 |
| MALDI | 軟 | 基質吸雷射再傳能量給樣品 | 強 | 否 | 大分子（蛋白質），田中耕一2002諾貝爾 |
| ESI | 軟 | 庫侖爆炸產生多電荷離子 | 強 | LC-MS | 多電荷！Fenn 2002諾貝爾 |
| DESI | 軟 | 電灑離子撞固體表面 | 強 | 否 | 不需前處理 |

### CI 四種反應（以 CH₄ 為試劑）

1. **質子轉移**：M + CH₅⁺ → [M+H]⁺ + CH₄
2. **加成**：M + C₂H₅⁺ → [M+C₂H₅]⁺
3. **氫離子轉移**：M + C₂H₅⁺ → [M-H]⁺ + C₂H₆
4. **電荷轉移**：M + CH₄⁺• → M⁺• + CH₄

質子親和力越高 → 越不容易轉移質子（H₂: 101 < CH₄: 127 < H₂O: 164 < i-C₄H₁₀: 195 < NH₃: 207）

## 質量分析器 (Mass Analyzers)

### 四種分析器比較

| 類型 | 原理 | 優點 | 缺點 |
|------|------|------|------|
| **磁扇區** | 磁場偏轉，m/z = B²r²e/2V | 高解析度、質量範圍大 | 速度慢、貴、大、不易串聯層析 |
| **四極柱** | DC+RF 動態電場過濾 | 體積小、便宜 | 質量範圍小、解析度一般 |
| **離子阱** | 射頻囚限離子 | 體積小、便宜、**可 MSⁿ** | 質量範圍小、解析度低、離子抑制 |
| **TOF** | 飛行時間差（輕快重慢） | 高解析度、質量範圍大、速度快 | 大、不易串聯層析 |

### 磁扇區公式推導

- 磁力 = 離心力：Bzev = mv²/r
- 整理得：**m/z = B²r²e / 2V**
- V↑ → 偵測到更小的 m/z；B↑ 或 r↑ → 偵測到更大的 m/z

### 雙焦點質譜儀

磁扇區 + 靜電場 (ESA) → 能量聚焦 + 動量聚焦 → 雙焦點 → 大幅提升解析度

## 質譜量測

### 基峰 (Base Peak)

- 最高訊號峰訂為 100%，其他峰以相對強度表示

### 加成物 (Adduct)

- [M+H]⁺（質子加成）、[M+Na]⁺（鈉加成）、[M-H]⁻（去質子）

### 特徵碎片模式

| 類型 | 特徵 m/z | 備註 |
|------|---------|------|
| 醛類 | **29**（必為基峰）| α-斷裂 |
| 甲基酮類 | **43** | 大烷基先斷 |
| 醇類 | **31** | 短鏈脫氫，長鏈脫水(-18) |
| 芳香族 | **91** | 草鎖離子 (tropylium) |
| 含 Cl | M, M+2 = **3:1** | ³⁵Cl:³⁷Cl |
| 含 Br | M, M+2 = **1:1** | ⁷⁹Br:⁸¹Br |

---

## 同位素 (Isotopes)

- Cl：³⁵Cl:³⁷Cl ≈ 3:1
- Br：⁷⁹Br:⁸¹Br ≈ 1:1
- 雙 Cl：(3+1)² = 9:6:1
- 三 Cl：(3+1)³ = 27:27:9:1
- 雙 Br：(1+1)² = 1:2:1
- 一 Cl 一 Br：(3+1)×(1+1) = 3:4:1

口訣：看到 M+2 很高 → 先想 Br 或 Cl！

---

## 解析度 (Resolution)

**R = m̄ / Δm**

- m̄：兩峰質量平均值，Δm：兩峰質量差
- 低解析：C₂H₄⁺、N₂⁺、CO⁺ 都在 m/z=28 無法區分
- 高解析：28.0313、28.0061、27.9949 可精確區分

Example：分離 C₂H₄⁺ (28.0313) 和 CH₂N⁺ (28.0187)
- Δm = 0.0126, m̄ = 28.0250
- R = 28.0250 / 0.0126 = 2.22×10³

## 串聯質譜 (MS/MS)

- **目的**：同分異構物無法用 MS 區分 → MS/MS 碎裂後可區分
- **CID**：碰撞誘導解離，能量越高碎片越多

### MSⁿ 層級

| 層級 | 說明 |
|------|------|
| MS | 離子化後直接分析 |
| MS² | 選母離子 → CID → 分析子離子 |
| MS³ | 子離子再碎裂 → 分析孫離子 |

### 四種掃描模式

| 模式 | 說明 |
|------|------|
| Product Ion Scan | 選定母離子，掃描所有子離子 |
| Precursor Ion Scan | 選定子離子，掃描所有母離子 |
| Neutral Loss Scan | 掃描失去固定 m/z 的母子離子對 |
| SRM | 只監測特定母→子離子反應 |

### SIM / SRM / MRM

| 模式 | 說明 |
|------|------|
| SIM | 只監測一個 m/z |
| SRM | 監測一對母→子離子 |
| MRM | 監測多對母→子離子 |
