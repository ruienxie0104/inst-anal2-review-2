🔬 Chapter 20 — 質譜法 (Mass Spectrometry)

一、質譜基本原理 ▶
質譜法 (Mass Spectrometry)

根據荷質比（mass-to-charge ratio, m/z）來分析離子的技術

A technique for analyzing ions based on their mass-to-charge ratio (m/z).

質譜不是「看光吸收」，而是打碎分子，秤碎片重量。

核心流程：帶電 → 加速 → 偏轉 → 偵測

歷史人物

  • Dalton（1803）：提出原子論，H = 1 amu，Da 單位由來
  • Cannizzaro（1860）：用 O = 16 amu 做標準
  • Aston（1919）：發明第一台質譜儀，1922 年諾貝爾化學獎

精確原子量

C = 12.0000, H = 1.0078, O = 15.9949, N = 14.0031

高解析質譜靠這些微小差異區分不同分子式

二、離子源 — 硬離子源 vs 軟離子源 ▶
硬離子源 (Hard Ionization)

能量大，碎片多，分子離子峰弱，但結構資訊豐富

High-energy ionization producing extensive fragmentation but rich structural information.
軟離子源 (Soft Ionization)

能量小，碎片少，分子離子峰強，但結構資訊少

Low-energy ionization producing intact molecular ions with minimal fragmentation.

口訣：硬碎結構清，軟整分子明

四種游離方式

  1. 電子捕捉：M + e⁻ → M⁻•
  2. 分解電子捕捉：AB + e⁻ → A⁻ + B•
  3. 分解離子化：AB + e⁻ → A⁺ + B⁻ + e⁻
  4. 產生離子對：AB + e⁻ → A⁺ + B⁻ + 2e⁻

1. EI 電子游離 (Electron Ionization)

  • 最經典的硬離子源，70 eV 電子束撞擊
  • 反應式：M + e⁻ → M⁺• + 2e⁻
  • 碎片多，結構資訊豐富，但分子離子峰可能很弱

2. CI 化學游離 (Chemical Ionization)

  • 軟離子源，先游離試劑氣體再撞擊樣品
  • 主要反應：質子轉移 CH₅⁺ + M → MH⁺ + CH₄
  • 分子離子峰出現在 M+1 位置

質子親和力：$$\text{NH}_3(207) > i\text{-C}_4\text{H}_{10}(195) > \text{H}_2\text{O}(164) > \text{C}_2\text{H}_6(159) > \text{CH}_4(127) > \text{H}_2(101)$$

3. FAB 快速原子撞擊 (Fast Atom Bombardment)

  • 軟離子源：快原子撞擊樣品表面使分子脫附
  • 適合極性大、熱不穩定分子（胜肽、醣類）
  • 樣品溶在甘油基質中

4. MALDI 基質輔助雷射脫附游離

  • 軟離子源：基質吸收雷射能量，溫和脫附樣品
  • 基質三要件：溶、吸、酸（互溶、吸雷射、酸性質子）
  • 田中耕一 — 2002 年諾貝爾化學獎
  • 特色：單電荷為主 [M+H]⁺，圖譜簡單

5. ESI 電灑游離 (Electrospray Ionization)

  • 軟離子源：帶電液滴 → 溶劑蒸發 → 庫侖爆炸 → 多電荷離子
  • 多電荷是最大特色：大分子 m/z 仍落在儀器範圍內
  • Fenn — 2002 年諾貝爾化學獎（與田中耕一同年）
  • 常與 LC 串聯（LC-MS）

6. DESI 脫附電灑游離

  • 軟離子源：ESI 產生的帶電液滴噴向固體表面
  • 免前處理，可直接分析固體表面
  • 環境游離的代表
三、質量分析器 ▶

1. 磁扇區 (Magnetic Sector)

  • 離子在磁場中偏轉，不同 m/z 偏轉半徑不同
  • 公式：$$\frac{m}{z} = \frac{B^2 r^2 e}{2V}$$
  • 優點：高解析度、質量範圍大
  • 缺點：昂貴、體積大、掃描慢

雙焦點質譜儀

ESA（靜電分析器）+ 磁場 + 狹縫 = 能量 + 方向雙重聚焦 = 高解析

2. 四極柱 (Quadrupole)

  • DC + RF 混合電壓，只有特定 m/z 能穩定通過
  • m/z 太小 → 震盪不穩定 → 撞極柱
  • m/z 剛好 → 穩定通過 → 偵測器
  • m/z 太大 → DC 推向外側 → 撞極柱
  • 優點：小、便宜、掃描快；缺點：質量範圍小、解析一般

3. 離子阱 (Ion Trap)

  • 離子被射頻電場困住，依序釋放不同 m/z
  • 最大特色：可做 MSⁿ（多次碎裂）
  • Dehmelt — 1989 年諾貝爾物理獎
  • 優點：小、便宜、靈敏度高

4. TOF 飛行時間 (Time of Flight)

  • 輕的離子飛得快，重的飛得慢，測到達時間反推 m/z
  • $$t \propto \sqrt{m/z}$$
  • 優點：高解析、質量範圍極大（理論無上限）、掃描極快
  • 缺點：體積大時不易串聯 MS/MS
分析器原理優點缺點特色
磁扇區磁場偏轉高解析、質量範圍大貴、大、慢$$m/z = B^2r^2e/2V$$
四極柱DC+RF 過濾小、便宜、快質量範圍小、解析一般質量濾波器
離子阱射頻囚禁小、便宜、可 MSⁿ質量範圍小、離子抑制MSⁿ
TOF飛行時間高解析、大範圍、極快大、不易串聯輕快重慢

口訣：磁扇區高準貴大，四極柱小便宜快，離子阱 MSⁿ 小靈敏，TOF 大範圍超快

四、正負離子模式與基峰 ▶

正負離子模式

  • 正離子模式：偵測 [M+H]⁺、[M+Na]⁺ 等
  • 負離子模式：偵測 [M-H]⁻ 等
  • 一次分析只能選一種模式！

基峰 (Base Peak)

基峰 (Base Peak)

質譜圖中訊號最強的峰，設為 100%

The most intense peak in a mass spectrum, set to 100% relative intensity.

基峰 ≠ 分子離子峰！基峰可能是碎片峰。

常見加成物：[M+Na]⁺、[M+K]⁺、[M+H-H₂O]⁺

五、特徵碎片與化合物判斷 ▶

四大特徵碎片口訣：醛 29、酮 43、醇 31、芳香族 91

醛類 — m/z = 29 基峰

  • m/z = 29（CHO⁺）必為基峰
  • [M-29]⁺：失去 CHO
  • [M-15]⁺：失去 CH₃

酮類 — 甲基酮基峰 m/z = 43

  • m/z = 43（CH₃CO⁺，乙醯基離子）基峰
  • 越大烷基越容易斷（α 斷裂）

羧酸與酯

  • 羰基兩側都可能斷裂
  • 酯類偏好斷 C-O 鍵（產生穩定醯基離子）

醇類 — m/z = 31

  • m/z = 31（CH₂OH⁺） 是醇的標誌碎片
  • 短鏈醇：脫氫 [M-1]⁺
  • 長鏈醇：脫水 [M-18]⁺

口訣：短醇脫氫(-1)，長醇脫水(-18)

芳香族 — m/z = 91 草鎖離子

  • m/z = 91（C₇H₇⁺，草鎖離子 Tropylium Ion）
  • 含苯環連一個碳的化合物都可能產生

醇類 EI 特徵

六、同位素判斷 ▶
同位素 (Isotope)

質子數相同、中子數不同的同一元素，質譜可區分

Atoms of the same element with different numbers of neutrons.
📌 鹵素同位素必背
  • Cl：³⁵Cl:³⁷Cl = 3:1，含一個 Cl → M : M+2 ≈ 3:1
  • Br：⁷⁹Br:⁸¹Br ≈ 1:1，含一個 Br → M : M+2 ≈ 1:1
  • 雙 Cl：M : M+2 : M+4 = 9:6:1

口訣：Cl 三比一，Br 一比一，雙 Cl 9:6:1

七、解析度與計算 ▶
解析度 (Resolution)

$$R = \frac{\bar{m}}{\Delta m}$$，分辨相近質量離子的能力

The ability to distinguish between two ions of similar mass.
$$R = \frac{\bar{m}}{\Delta m}$$
$$\bar{m}$$ = 兩峰質量平均值 $$\Delta m$$ = 兩峰質量差
八、串聯質譜 (MS/MS) ▶

MS/MS 解決同分異構物問題：分子式相同但結構不同

CID 碰撞誘導解離

三重四極柱：$$\text{Q}_1$$（選）→ $$\text{Q}_2$$（撞）→ $$\text{Q}_3$$（析）

MSⁿ

離子阱可做 MSⁿ（反覆捕捉碎裂），三重四極柱只能做 MS²

四種掃描模式

模式固定掃描問題
Product Ion Scan母離子 m/z所有子離子這離子碎成什麼？
Precursor Ion Scan子離子 m/z所有母離子誰碎出這個碎片？
Neutral Loss Scan丟失質量符合的母離子誰丟了這個質量？
SRM/MRM母→子 m/z 對無特定轉換，最靈敏

口訣：子離子掃描看碎什麼，母離子掃描找誰碎的，中性丟失看丟什麼，SRM 看特定轉換

碰撞能量影響

低能量 → 分子離子峰強、碎片少；高能量 → 碎片多、分子離子峰弱

⚡ Chapter 22-25 — 電分析化學 (Electroanalytical Chemistry)

一、氧化還原與電化學電池 ▶
氧化 (Oxidation)

失去電子，氧化數增加

LEO — Lose Electrons = Oxidation
還原 (Reduction)

得到電子，氧化數減少

GER — Gain Electrons = Reduction

口訣：LEO says GER 🔥

電化學電池

電極發生什麼記法
陰極 Cathode還原反應（得到電子）Cat 還
陽極 Anode氧化反應（失去電子）An 氧

Plus Right Rule

$$E_{\text{cell}} = E_{\text{right}} - E_{\text{left}}$$

正極永遠接右邊的半電池

電池表示法

$$\text{Cu} \, | \, \text{Cu}^{2+}(0.02\text{M}) \, || \, \text{Ag}^+(0.02\text{M}) \, | \, \text{Ag}$$

左氧右還：左邊是陽極（氧化），右邊是陰極（還原）

二、標準電極電位 ▶
標準氫電極 (SHE)

所有電極電位的基準，E° = 0.000 V（所有溫度）

The universal reference for electrode potentials, defined as 0.000 V at all temperatures.
📌 標準電極電位五大重點
  1. 相對 SHE（0.000 V）決定
  2. 為還原電位（表列值都是還原反應）
  3. 標準狀態下得出（1 M, 1 atm, 25°C）
  4. 與反應莫耳數無關（E° 不因計量係數改變！）
  5. 正電位 = 自發反應

口訣：相對SHE、還原電位、標準狀態、莫耳無關、正=自發

三、Nernst 方程式 ▶

完整版

$$E = E° - \frac{RT}{nF} \ln Q$$

25°C 簡化版

$$E = E° - \frac{0.0592}{n} \log Q$$
$$E$$：非標準狀態電極電位 $$E°$$：標準電極電位 $$n$$：轉移電子數 $$Q$$：反應商（產物/反應物）

三個 Nernst 範例

  1. $$\text{Zn}^{2+} + 2e^- \rightleftharpoons \text{Zn}$$：$$E = E° + \frac{0.0592}{2}\log[\text{Zn}^{2+}]$$
  2. $$\text{Fe}^{3+} + e^- \rightleftharpoons \text{Fe}^{2+}$$：$$E = E° - 0.0592\log\frac{[\text{Fe}^{2+}]}{[\text{Fe}^{3+}]}$$
  3. $$2\text{H}^+ + 2e^- \rightleftharpoons \text{H}_2$$：$$E = E° - \frac{0.0592}{2}\log\frac{P_{\text{H}_2}}{[\text{H}^+]^2}$$

Example 22-2 — Cd²⁺ 非標準濃度

$$[\text{Cd}^{2+}] = 0.0150\text{ M}$$，求電極電位

$$E = -0.403 - \frac{0.0592}{2} \log\frac{1}{0.0150} = -0.457\text{ V}$$

濃度越低，電位越負（越不利於還原）

Example A — Ag/AgCl 電極電位

$$K_{sp}(\text{AgCl}) = 1.82 \times 10^{-10}$$，$$[\text{Cl}^-] = 1\text{ M}$$

$$[\text{Ag}^+] = \frac{K_{sp}}{[\text{Cl}^-]} = 1.82 \times 10^{-10}\text{ M}$$

$$E = 0.799 + 0.0592 \times \log(1.82 \times 10^{-10}) = 0.222\text{ V}$$

五、伏安法家族 ▶
伏安法 (Voltammetry)

施加時變電位（E = f(t)），同時量測電流的方法

A method applying time-varying potential and measuring current simultaneously.

LSV 線性掃描伏安法

  • 電位線性掃描，同時量測電流
  • 峰電流 i_p → 定量（濃度越高，峰越高）
  • 峰電位 E_p → 定性（不同物質在不同電位出峰）

CV 循環伏安法

  • 三角波掃描：先氧化方向，再還原方向
  • 可同時看到氧化峰和還原峰
  • 可逆：對稱雙峰，$$\Delta E_p \approx \frac{59}{n}\text{ mV}$$
  • 準可逆：峰分離變大
  • 不可逆：只有一個方向的峰

口訣：可逆 = 對稱雙峰、E° 取中間

Randles-Ševčík 方程式

$$i_p = (2.69 \times 10^5) \cdot n^{3/2} \cdot A \cdot C \cdot D^{1/2} \cdot v^{1/2}$$
$$n$$：轉移電子數 $$A$$：電極面積 (cm²) $$C$$：濃度 (mol/cm³) $$D$$：擴散係數 $$v$$：掃描速率 (V/s)

$$i_p \propto C$$（定量基礎）、$$i_p \propto v^{1/2}$$（掃越快峰越高）

DPV 與 SWV

  • DPV（微分脈衝伏安法）：疊加脈衝，取電流差消除充電電流
  • SWV（方波伏安法）：方波波形，比 CV 訊雜比更高、更快
六、定電位安培法 ▶
  • 固定電位，量測電流隨時間變化（i-t curve）
  • 電流一開始大，隨時間遞減，最終趨近穩定值
  • 選擇性高：選對電位只量測目標反應

口訣：定電位 = 選對頻道聽特定電台

電位選擇 — 多成分分析

施加電位可偵測的物質
−0.1 VCu
−0.5 VCu + Pb
−0.6 VCu + Pb + Cd
−0.8 VCu + Pb + Cd + Ga
−1.0 V全部 + Zn

電位越負，偵測越多；低電位干擾高電位

🔢 公式速查

Ch20 質譜法

離子加速動能

$$KE = zeV$$
$$z$$ = 電荷數 $$e = 1.6 \times 10^{-19}\text{ C}$$ $$V$$ = 加速電壓

磁扇區質譜儀

$$\frac{m}{z} = \frac{B^2 r^2 e}{2V}$$
$$B$$ = 磁場強度 (T) $$r$$ = 曲率半徑 (m) $$e = 1.6 \times 10^{-19}\text{ C}$$ $$V$$ = 加速電壓 (V)

磁扇區加速電壓

$$V = \frac{B^2 r^2 z e}{2m}$$
$$m$$ 用 kg（$$\text{amu} / 6.02 \times 10^{23}$$） $$r$$ 用 m（cm / 100）

解析度

$$R = \frac{\bar{m}}{\Delta m}$$
$$\bar{m}$$ = 兩峰質量平均值 $$\Delta m$$ = 兩峰質量差

同位素比例

$${}^{35}\text{Cl} : {}^{37}\text{Cl} = 3:1 \Rightarrow M:M+2 \approx 3:1$$
$${}^{79}\text{Br} : {}^{81}\text{Br} \approx 1:1 \Rightarrow M:M+2 \approx 1:1$$
雙 Cl: $$M:M+2:M+4 = 9:6:1$$

Ch22-25 電分析化學

電池電位

$$E_{\text{cell}} = E_{\text{right}} - E_{\text{left}}$$

Nernst 方程式 (25°C)

$$E = E° - \frac{0.0592}{n} \log \frac{[\text{C}]^c [\text{D}]^d}{[\text{A}]^a [\text{B}]^b}$$
$$E°$$ = 標準電極電位 $$n$$ = 轉移電子數 固體和純液體濃度 = 1 0.0592 只在 25°C 適用

Ksp 結合 Nernst

$$E = E° + \frac{0.0592}{n} \log \frac{K_{sp}}{[\text{Cl}^-]}$$

（以 Ag/AgCl 為例）

CV 可逆反應

$$E° = \frac{E_{\text{氧化峰}} + E_{\text{還原峰}}}{2}$$
$$\Delta E_p \approx \frac{59}{n}\text{ mV} \quad (25°C)$$

Randles-Ševčík

$$i_p = (2.69 \times 10^5) \cdot n^{3/2} \cdot A \cdot C \cdot D^{1/2} \cdot v^{1/2}$$
$$n$$ = 轉移電子數 $$A$$ = 電極面積 (cm²) $$C$$ = 濃度 (mol/cm³) $$D$$ = 擴散係數 (cm²/s) $$v$$ = 掃描速率 (V/s)

參考電極電位

$$\text{SHE}: E° = 0.000\text{ V} \quad \text{SCE}: E° = +0.244\text{ V} \quad \text{Ag/AgCl}: E° = +0.197\text{ V}$$

🎯 考題預測與解題策略

Ch20 質譜法

🔴 必考題型

1. 離子源選擇

給分析需求選最適合的離子源：

  • 大分子蛋白質 → ESI 或 MALDI
  • 需要結構資訊 → EI
  • 只需分子量 → CI、ESI、MALDI
  • GC-MS → EI 或 CI
  • LC-MS → ESI
  • 不需前處理 → DESI
2. 磁扇區計算

給 B, r, V, m/z 中的三個求第四個

公式：$$\frac{m}{z} = \frac{B^2 r^2 e}{2V}$$

⚠️ 注意單位：r 要換算成 m，m 要換算成 kg

3. 解析度計算

給兩個離子的精確質量，求 $$R = \frac{\bar{m}}{\Delta m}$$

⚠️ m̄ 是平均值，不是任一個！

4. 同位素判斷
  • M+2 ≈ M/3 → 一個 Cl
  • M+2 ≈ M → 一個 Br
  • M:M+2:M+4 = 9:6:1 → 兩個 Cl
5. 特徵碎片辨識
  • m/z=29 基峰 → 醛
  • m/z=43 基峰 → 甲基酮
  • m/z=31 → 醇
  • m/z=91 → 芳香族（草鎖離子）
6. 串聯質譜掃描模式
  • 想知道某分子的碎片 → Product Ion Scan
  • 想知道哪些分子會產生某碎片 → Precursor Ion Scan
  • 想找失去特定質量的分子對 → Neutral Loss Scan
  • 想定量特定反應 → SRM/MRM

⚠️ 易錯點

  • E° 是還原電位，不是氧化電位
  • 解析度公式用平均值 $$\bar{m}$$，不是 $$m_1$$ 或 $$m_2$$
  • 磁扇區公式中 $$r$$ 要用公尺，$$m$$ 要用公斤
  • 同位素比例：$$(3+1)^n$$ 是 Cl 的，$$(1+1)^n$$ 是 Br 的

Ch22-25 電分析化學

🔴 必考題型

1. E° 正負判斷

正電位 = 自發（還原反應），負電位 = 非自發

2. Nernst 計算

給非標準濃度求 E

  • 注意 n 值（轉移電子數）
  • 注意 log 裡的分母分子
  • 0.0592 只在 25°C 適用
3. Ksp + Nernst 結合

關鍵：[Ag⁺] = Ksp / [Cl⁻]

4. Cell Potential 計算

$$E_{\text{cell}} = E_{\text{right}} - E_{\text{left}}$$

5. CV 圖判讀
  • 可逆：對稱，$$\Delta E_p \approx \frac{59}{n}\text{ mV}$$
  • 準可逆：峰分離變大
  • 不可逆：只有一個方向的峰

⚠️ 易錯點

  • Plus Right Rule：電錶正極接右邊，$$E_{\text{cell}} = E_{\text{right}} - E_{\text{left}}$$
  • E° 與反應莫耳數無關！
  • Nernst 的 0.0592 只在 25°C 適用
  • 固體和純液體濃度 = 1（不寫在 Q 裡）
  • SCE 用汞，有環境毒性問題
  • 電位選擇：高電位會受低電位干擾
